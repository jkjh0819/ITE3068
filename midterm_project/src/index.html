D3.js와 Billboard.js를 사용하여 차트를 만들어보고
관련 설명을 담고 있는 페이지입니다.

데이터 셋은 Kaggle에서 제공하는 Titanic: Machine Learning from Disaster train set을 활용하였습니다. 같은 데이터에 대해 D3.js와 Billboard.js를 이용하여 여러 종류의 차트를 만들어보고 사용한 요소에 대해 설명을 서술하였습니다.
자세한 설명은 wiki를 참조하시기 바랍니다.
